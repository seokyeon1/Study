# 자료 구조화

**종류** : 리스트, 스택, 큐, 테크 (선형 구조)/ 트리, 그래프(비선형 구조)

##### 표현

- 순차 표현 : 배열을 사용한다. 논리 구조 = 물리 구조
- 연결 표현 : 포인터를 사용한다. 논리 구조 ≠ 물리 구조



##### 리스트 삽입

##### 1. 마지막에 삽입될 때

```
1. new 노드 생성
2. new 노드에 값 입력
3. 링크 값이 null 인 노드 찾기 (temp)
4. temp 의 link 값을 new
```



##### 2. 맨 앞에 삽입될 때

```
1. new 노드 생성
2. new 노드에 값 입력
3. new 의 link head 값으로
4. head 값을 new
```



##### 3. 중간에 삽입될 때

```
1. new 노드 생성
2. new 노드에 값 입력
3. new 의 link 를 pre 의 link 값으로
4. pre 의 link 를 new로
```
